# Library Manager

A simple **Command-Line Library Management System** using Python, designed for campus libraries to manage books efficiently.

## Features
- Add new books to the library.
- Issue and return books.
- Search books by title or ISBN.
- View all books in the library.
- Persistent storage using JSON.
- Logging of important actions and errors.
